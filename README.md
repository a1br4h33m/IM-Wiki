# IM-Wiki

## Om prosjektet
Dette prosjektet er en wiki for **IM (Informasjonsteknologi og Medieproduksjon)**. Wikien skal brukes til å samle informasjon om fagstoff, teknologier og ulike IT-temaer.

## Oppgave
Målet med oppgaven er å lage **wiki-systemet**, altså strukturen og funksjonene. 
Lærerne ønsket også at systemet skal gjøre det **enkelt å legge til nye sider med temaer**, slik at wikien kan utvides over tid.

## Mulige temaer
- Programmering  
- Nettverk  
- Operativsystemer  
- Webutvikling  
- IT-sikkerhet  

## Teknologier
Prosjektet skal bruke teknologier som:
- HTML  
- CSS  
- JavaScript
- Python
- Flask
- MariaDB
